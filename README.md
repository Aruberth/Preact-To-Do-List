## Instalação
- **Clone o projeto**
  ```
  git clone https://github.com/Aruberth/Preact-To-Do-List.git
  cd nome-do-projeto
  ```
- **Instale as dependências**
  ```
  npm install
  ```
## Uso Geral
- **Execute o comando no terminal:**
  ```
  npm run dev
  ```
